# 🚀 TalentFlow — Recruitment Management Platform

TalentFlow is a modern recruitment management application built with **Symfony 7** (API Platform) and **Vue 3** (TypeScript, Pinia, PrimeVue).

## 🛠 Tech Stack

- **Backend:** PHP 8.3, Symfony 7, API Platform 3, PostgreSQL, LexikJWTAuthenticationBundle
- **Frontend:** Vue 3, TypeScript, Pinia, Vite, PrimeVue, Axios
- **DevOps:** Docker, Docker Compose, Nginx

## 🏗 Architecture

The project is structured as a monorepo:

- `/backend`: Symfony 7 API & Business Logic
- `/frontend`: Vue 3 SPA Framework
- `/docker`: Infrastructure configuration

## 🚀 Getting Started

### Prerequisites

- Docker & Docker Compose
- Node.js >= 20 (for local frontend dev)
- PHP >= 8.3 & Composer (for local backend dev)

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/vencesla/talentflow.git](git@github.com:vencesla/talentflow.git)
   cd talentflow
   ```
